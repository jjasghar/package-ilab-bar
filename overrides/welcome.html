ilab-bar 0.0.1

This installer will guide you through the steps necessary to install ilab-bar project.

Our goal for this project is to have an easy way to verify engaging with the models via ilab.

Please take a moment to look at the Getting Started Guide and if you have any questions reach out to #users on our Public Community resources.

To start this application you'lle need to run ilab-bar on the command line, this also assumes you have installed the packaged verison of instructlab, otherwise you'll need to create to add ilab to your $PATH.
